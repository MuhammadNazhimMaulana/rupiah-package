# Rupiah Package

This package is used for converting number to rupiah format

Do not forget to run this command after cloning:

```
composer install
```

To get list of available test group use this command:

```
./vendor/bin/phpunit --list-groups
```

After getting list of group you could run it with this command

```
./vendor/bin/phpunit --group groupname
```
